# Guia Orientativo para Servidores - PICTI

Guia institucional sobre submissão e execução do PICTI para servidores do Ifes Campus Colatina.

Desenvolvido por Luisa Muylaert de Menezes Póvoa, da Coordenadoria de Pesquisa do Ifes - Campus Colatina, 2026.

## Arquivos

- `index.html`: página publicada pelo GitHub Pages.
- `guia-picti-servidores.html`: arquivo-fonte do guia.
- `colatina-horizontal-cor.jpg`: logotipo utilizado no guia.

## Publicação

O `index.html` deve permanecer sincronizado com `guia-picti-servidores.html`.
Arquivos temporários de testes de impressão e o pacote ZIP local não devem ser enviados ao repositório.
